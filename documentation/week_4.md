Week 4 - GitHub Project Management, Data Cleaning Techniques & KPI Introduction

Overview
The fourth week focused on improving our project organization using GitHub and strengthening our understanding of data cleaning concepts. The mentor also introduced basic concepts of Exploratory Data Analysis (EDA) and Key Performance Indicators (KPIs), which are essential for building meaningful dashboards and analyzing business performance.

Day-wise Learning Summary

Monday to Thursday: Learning Sessions
• The mentor explained the importance of maintaining a GitHub repository to document and showcase our project work.  
• We were instructed to upload our datasets, documentation, and project files regularly so that GitHub can serve as a portfolio of our work.  
• During the data cleaning discussion, we learned how to handle missing values in numerical columns.  
• Null values in numerical columns can be replaced using the average (mean) of the column to maintain data consistency.  
• The mentor introduced the concept of Exploratory Data Analysis (EDA) and explained how it helps in understanding the structure of a dataset and identifying patterns.  
• The concept of Key Performance Indicators (KPIs) was also explained, including their role in measuring performance and tracking important metrics in a project or business environment.

Friday: Review & Repository Presentation
• A review session was conducted for all the team members.  
• Each member introduced themselves and shared what they had learned during the internship so far.  
• Everyone presented their GitHub repositories and explained the structure of their project folders, datasets, and documentation.

Key Outcomes
• Improved understanding of maintaining project documentation using GitHub.  
• Learned methods to handle missing values in numerical data.  
• Gained basic knowledge of Exploratory Data Analysis (EDA).  
• Understood the importance and use of Key Performance Indicators (KPIs).  
• Developed confidence in presenting project work and explaining repository structure.
