# Optimizing IT Support Team Performance Using Analytics

## 1. Problem Statement

IT support teams manage a large number of service tickets on a daily basis, making it difficult to track performance, response efficiency, and workload distribution. Delays in issue resolution and uneven ticket allocation can reduce overall service quality. This project focuses on analyzing IT support data to identify operational gaps and improve efficiency. By cleaning and transforming raw data using Excel and Power Query, the objective is to prepare structured data for meaningful performance analysis. The ultimate goal is to support better decision-making through data-driven insights.


## 2. Dataset Description

The dataset used in this project contains IT service desk ticket information. It includes details related to ticket creation, assignment, priority level, resolution time, and status tracking.

Key fields in the dataset include:

- Ticket ID  
- Issue Category  
- Priority Level (Low, Medium, High, Critical)  
- Assigned Support Agent  
- Ticket Status (Open, In Progress, Closed)  
- Created Date  
- Resolution Date  
- Resolution Time  
- Customer Satisfaction Rating  

The dataset initially contains raw and uncleaned data, including missing values and inconsistencies. Data preprocessing and transformation are performed using Excel and Power Query to ensure accuracy and reliability for further analysis.
